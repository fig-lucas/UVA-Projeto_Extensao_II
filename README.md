# UVA-Projeto_Extensao_II
Trabalho de Extensão - UVA

Software em C para gestão de vendas e fechamento de caixa de microempresas.

O sistema realiza o cadastro de vendas, aplica regras de negócio para devoluções, gera relatórios periódicos e ordena o faturamento anual utilizando o algoritmo Bubble Sort. Desenvolvido de forma modular com uso de structs, ponteiros e gerenciamento eficiente de memória para garantir estabilidade e leveza na execução.
